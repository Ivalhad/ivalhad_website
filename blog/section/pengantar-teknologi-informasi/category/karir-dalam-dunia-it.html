Front-end Website Developer

Dalam pengembangan website front end web developer adalah seorang yang bertanggung jawab dalam pengembangan website dari segi tampilan dan berinteraksi langsung dengan client atau pengguna. Oleh karena itu nama lain dari front end web dev adalah client side. Kemampuan atau skill yang harus dimiliki seorang front end web dev dari tinggkat pemula hingga tinggkat lanjut berdasarkan standar global.

A. Tingkat Pemula

  1. 1. Pengenalan mengenai website, apa itu website, tools dalam mengembangkan sebuah website.
  2. 2. Fungsi HTML dalam website, memahami struktur dasar HTML
  3. 3. Mengorganisir konten dengan elemen semantik yang ada, mengenal attribute pada HTML, mengenal paragraf, header, gambar, list, dan elemen teks lainnya. Memahami perbedaan inline dan block elemen.
  4. 4. Memahami peran CSS dalam pembuatan website, benefit menerapkan CSS dan mengetahui penulisan sintaks dasar CSS dengan menerapkan beberapa basic styling pada elemen HTML.
  5. 5. Memahami tentang penggunaan flexbox sebagai layout dan dapat dikombinasikan dengan media query agar lebih optimal ketika diakses menggunakan mobile devices.
  6. 6. Memahami peran JavaScript dalam pengembangan website, menuliskan sintaks dasar JavaScript, memahami dasar JavaScript seperti statement, comment, variabel, tipe data, dsb. Mengetahui Browser Object dan event yang ada, dan memanipulasi DOM menggunakan JavaScript.

A. Tingkat Menengah

  1. 1. Menerapkan sintaks terbaru pada JavaScript seperti block scope variable, template literals, destructuring object and array, spread operator dan rest parameters, arrow function, classes, promise hingga JavaScript module.
  2. 2. Membangun component UI website yang bersifat reusable dan encapsulated dengan custom element dan shadow DOM.
  3. 3. Menambahkan, menghapus, dan menggunakan package JavaScript yang ada di NPM, serta membedakan fungsi dari development dependency dan production dependency.
  4. 4. Menggunakan webpack sebagai environment dan build tools dalam membangun website yang optimal.
  5. 5. Dapat melakukan transaksi data melalui protokol HTTP/HTTPS menggunakan fetch API baik pada method GET, POST, PUT, dan DELETE.

A. Tingkat Lanjut

  1. 1. Memahami pendekatan mobile-first, serta belajar membangun website yang responsif menggunakan CSS Grid System, Media Query, dan Off Screen Canvas melalui pendekatan Mobile First.
  2. 2. mengetahui aspek aksesibilitas web seperti focus, semantic HTML, tab order, label, alternative teks, serta penggunaan screen reader agar website yang dikembangkan dapat diakses oleh siapa pun.
  3. 3. Mengetahui prinsip-prinsip penulisan kode JavaScript dengan benar berdasarkan senior software engineer.
  4. 4. Dapat membuat website yang progressive sehingga website memiliki sifat native seperti installable, offline capability, dan lainnya.
  5. 5.Dapat melakukan penerapan testing seperti Unit Test, Integration Test, dan End to End Test pada website yang dibuat, serta belajar menggunakan DevTools untuk debugging.
  6. 6. Dapat menggunakan web auditor untuk mengetahui sejauh mana performa website yang dibuat, dan mengoptimalkan performa website berdasarkan hasil yang diperoleh.
  7. 7. Dapat mendeploy aplikasi ke production secara otomatis, aman, dan cepat menggunakan teknik Continuous Integration dan Continuous Deployment.

Di tengah layoff yang terjadi di beberapa perusahaan startup, kebutuhan akan talenta digital di Indonesia tetap tinggi seiring dengan tren digitalisasi dan otomasi di setiap industri. Dengan banyaknya perusahaan yang mulai bertransformasi ke ranah digital, kesempatan untuk memiliki karir profesional di bidang digital semakin terbuka lebar. Salah satu keahlian yang banyak dicari adalah web developer atau pengembang website. Hampir semua situs pencari kerja selalu menampilkan lowongan untuk profesi web developer yang disebabkan oleh maraknya perusahaan yang merambah ke dunia digital. Karena kebutuhan yang cukup besar, menjadi web developer pun menjadi pilihan karir yang menarik dan bisa dipertimbangkan.
